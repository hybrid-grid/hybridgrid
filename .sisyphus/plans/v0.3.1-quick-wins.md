## ✅ Feature 1: --no-fallback Flag — ALREADY IMPLEMENTED

**Location**: `cmd/hgbuild/main.go`

**Evidence**:
- Line 165: `rootCmd.PersistentFlags().BoolVar(&noFallback, "no-fallback", false, "disable local fallback when coordinator is unavailable")`
- Lines 1288-1292: Logic that returns error when flag is set and coordinator unavailable

**Verification**:
```bash
# Without --no-fallback (falls back)
./bin/hgbuild --coordinator=localhost:9000 cc -c test.c
# Output: "Warning: coordinator not available, compiling locally"
# Exit: 0, object file created

# With --no-fallback (fails fast)
./bin/hgbuild --no-fallback --coordinator=localhost:9000 cc -c test.c
# Output: "Error: coordinator unavailable and fallback disabled"
# Exit: non-zero, no object file
```
