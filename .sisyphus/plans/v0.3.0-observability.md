**Effort**: 30 minutes

#### Subtasks
- [ ] **2.1.1**: Fix exit code handling (20min)
  - **File**: `test/stress/run-test.sh`
  - **Problem**: Lines 83, 99, 144 pipe to tail, masking exit codes
  - **Solution**:
    ```bash
    # Replace: make -j4 2>&1 | tail -10
    # With:
    TMPLOG=$(mktemp)
    make -j4 > "$TMPLOG" 2>&1
    EXIT_CODE=$?
    tail -10 "$TMPLOG"
    if [ $EXIT_CODE -ne 0 ]; then
        print_error "Build failed with exit code $EXIT_CODE"
        exit $EXIT_CODE
    fi
    rm "$TMPLOG"
    ```
  - **Apply to**: Local build (line 83), distributed build (line 99), cache test (line 144)
